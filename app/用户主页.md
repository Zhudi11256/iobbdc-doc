# 主页
## 主页
- `/api/user/poofile`
- `GET`
- `request`

| 参数名称 | 数据类型 | 是否必须 |参数说明|默认值|参数示例|
| :-----| :---- | :---- | :---- | :---- | :---- |

- `resonse`
```json
{
    "message": "ok",
    "success": true,
    "code": 0,
    "data": {
        "user": {
            "id": 1,                // 用户id
            "vipExpire": "2020-07-25 11:55:36", // vip过期时间
            "balance": "225.6",         // 账户余额
            "shareCount": 15,           // 推广人数
            "phone": "185****1565",     // 手机号码
        },
        "dailyPlayCount": 1,            // 今日播放
        "maxPlayCountOfDay": 20,        // 每日最多播放
        "dailyDownloadCount": 52,          // 今日下载
        "maxDownloadCountOfDay": 223        // 每日最多下载
    }
}
```
> 如果没data字段，表示用户未登录。所有数据显示默认值
