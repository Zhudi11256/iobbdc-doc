# 通知
## 启动弹窗通知
- `/api/notice`
- `GET`
- `request`

| 参数名称 | 数据类型 | 是否必须 |参数说明|默认值|参数示例|
| :-----| :---- | :---- | :---- | :---- | :---- |
|lastModifiedDate|string|否|最一次修改时间|||

- `resonse`
```json
{
    "message": "ok",
    "succes": true,
    "code": 0,
    "data": {
        "id": 1,
        "title": "",            // 标题
        "content": "",          // 内容
        "createdDate": "",      // 创建时间
        "lastModifiedDate": "", // 最后修改时间
    }
}
```
> 通知数据的最后修改时间，`lastModifiedDate` 优先 `createdDate`。
